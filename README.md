# Laravel Design Slider

Range slider component for Laravel. Supports Livewire, Blade, and Vue 3.

## Installation

```bash
composer require mrshanebarron/slider
```


## Usage

### Livewire Component

```blade
<livewire:ld-slider />
```

### Blade Component

```blade
<x-ld-slider />
```

## Configuration

Publish the config file:

```bash
php artisan vendor:publish --tag=ld-slider-config
```

## Customization

### Publishing Views

```bash
php artisan vendor:publish --tag=ld-slider-views
```
